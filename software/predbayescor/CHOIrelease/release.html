Bayesian Classification with high order interactions

Longhai Li, Department of Statitics, University of Toronto

Description

This software predicts the response with categorical features by learning from training data. We take the high-order interactions into account. We use a compression technique to reduce the number of parameters associated with high-order interactions. The model is trained with a sort of Markov chain sampling method, called slice sampling, which is developed by Professor Radford Neal in University of Toronto.

Install

The software is written in C for Unix System and uses R as user interface. R is a free software environment for statistical computing and graphics, available from http://www.r-project.org/.

If you have installed R in your computer, then download the add-on R package: ClassHighOrderInteract_1.0.tar.gz to a local directory. If you'd like to read the orginal .R and .c files download this tar file: ClassHighOrderInteract.ori.tar

To install this package in your computer to somewhere, say "myRlib", type the following command in Unix console:

$ R CMD INSTALL -l myRlib ClassHighOrderInteract_1.0.tar.gz

Load the library

Type the following command in R console you start up in whichever directory to load the package

> library("ClassHighOrderInteract", lib.loc="myRlib")

Help Documentation

After you have installed and libraried the package, in R console, type

> ?comp_train_pred_interact

R will show up the documentation as shown by help file in html format.
