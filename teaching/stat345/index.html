STAT 345 Introduction to Statistical Methods

Longhai Li, Department of Mathematics and Statistics, University of Saskatchewan

An introduction to the principles of experimental design and analysis of variance. Topics includes: randomization, blocking, factorial experiments, confounding, random effects, analysis of covariance. Emphasis will be on fundamental principles and data analysis techniques using R rather than on mathematical theory. The following is a list of demonstration HTML produced with Rmarkdown.

  1. Introduction and R Basics (introduction to statistics, R, import data into R)

  2. Analysis of Simple Comparative Experiments (paired t-test)

Rmarkdown and other supporting files for producing the above html files can be found from this Github folder.

Some examples are taken from the textbook Design and Analysis of Experiments by Douglas C. Montgomery.
