STAT 245 Introduction to Statistical Methods

Longhai Li, Department of Mathematics and Statistics, University of Saskatchewan

This course is an introduction to basic statistical methods including frequency distributions, elementary probability, confidence intervals and tests of significance, analysis of variance, regression and correlation, contingency tables, and goodness of fit. The following is a list of demonstration html files organized by the topics.

  1. Introduction of statistics and R Basics (introduction to statistics, R, import data into R) (Chapter 1)

  2. Frequency Graph (barplot, histogram, ECDF, stem-leaf, scatterplot) (Chapter 2)

  3. Numerical Summary (mean, median, sd, quantiles, boxplot) (chapter 3)

  4. Probability Theory (Chapter 4,5,6)

    • Binomial Distribution

    • Normal Distribution

  1. Sampling distribution (Central limit theory, Simulation to visualize CLT) (Chapter 7)

  2. Confidence Interval and Hypothesis Test for One-sample (z test, and t-test) (Chapter 8, 9)

  3. Confidence Interval and Hypothesis Test for Two-sample (z test, two-sample t-test, paired t-test) (Chapter 10)

  4. Analysis of Variance (Chapter 11)

  5. Chi-square test (for one or two categorical variables) (Chapter 12)

  6. Correlation and Regression (correlation, least square estimate, t-test, ANOVA, F-test for LSE) (Chapter 13)

As time permits, we will additionally talk about:

  1. Regression with a Categorical Variable (Dummy Variable)

  2. Multiple Regression

Rmarkdown and other supporting files for producing the above html files can be found from this Github folder.
