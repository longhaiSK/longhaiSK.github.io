STAT 245 Introduction to Statistical Methods

This course is an introduction to basic statistical methods including frequency distributions, elementary probability, confidence intervals and tests of significance, analysis of variance, regression and correlation, contingency tables, and goodness of fit. The following is a list of demonstration html files organized by the topics.

Rmarkdown and other supporting files for producing the above html files can be found from this Github folder. The chapter numbers refer to the numbers in the textbook Introductory Statistics, 10th Edition, By Prem S. Mann

  1. Introduction of statistics and R Basics (introduction to statistics, R, import data into R) (Chapter 1)

  2. Frequency Graph (barplot, histogram, ECDF, stem-leaf, scatterplot) (Chapter 2)

  3. Numerical Summary (mean, median, sd, quantiles, boxplot) (chapter 3)

  4. Probability Theory (Chapter 4,5,6)

    1. Binomial Distribution

    2. Normal Distribution

  5. Sampling distribution (Central limit theory, Simulation to visualize CLT) (Chapter 7)

  6. Confidence Interval and Hypothesis Test for One-sample (z test, and t-test) (Chapter 8, 9)

  7. Confidence Interval and Hypothesis Test for Two-sample (z test, two-sample t-test, paired t-test) (Chapter 10)

  8. Analysis of Variance (Chapter 12)

  9. Chi-square test (for one or two categorical variables) (Chapter 11)

  10. Correlation and Regression (correlation, least square estimate, t-test, ANOVA, F-test for LSE) (Chapter 13)

  11. Regression with a Categorical Variable (Dummy Variable)

  12. Multiple Regression (Chapter 14)
